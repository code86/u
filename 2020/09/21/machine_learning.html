Linear Regression

Let us first review linear regression.

Your exercise is to repeat the following analysis using a neural network.

import numpy as np
import pandas as pd

df1 = pd.read_stata('http://www.stata-press.com/data/r11/auto.dta')

from sklearn import linear_model as lm

X = df1[['length']].to_numpy()
Y = df1[['mpg']].to_numpy()
lm = lm.LinearRegression()
lm.fit(X, Y)
Y_pred = lm.predict(X)
[lm.intercept_, lm.coef_]

[array([60.15586182]), array([[-0.20676881]])]

The above result (negative slope, of -0.20676881) confirms our intuition that fuel efficiency ("mpg") is lower for longer vehicles ("length").

Now, let's check out the scatter plot of mpg versus length, as well as the fitted/predicted regression line.

import matplotlib.pyplot as plt

plt.scatter(X, Y)
plt.xlabel('length')
plt.ylabel('mpg')
plt.plot(X, Y_pred, color='red')
plt.show()

Objective

Replicate the results of the above linear regression, $mpg = w_0 + w_1 \cdot length$, using an artificial neuron (which is the basic element of a neural network).

$w_0$ and $w_1$ are generally known as the bias and weight respectively.

For the neural network to have the same intercept and cofficient as in linear regression: No hidden layer. No activation function. No data normalization. Use entire dataset for training (i.e., do not separate dataset into training, validation, and testing).

Your Task:

  1. Replicate the above linear regression with Tensorflow.Keras. This example may be helpful to get you started. In your trained/fitted model, the final weight and bias should be equal to the above (i.e., weight of -0.20676881, and bias of 60.15586182). Check if it is so.

  2. Repeat with sklearn.MLPRegressor. As before, no hidden layer / activation function. Use "linear / identity" for the activation function if it must be specified. "linear / identity" means that $f(x) = x$.

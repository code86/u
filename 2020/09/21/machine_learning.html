Linear Regression

Let us first review linear regression.

Your exercise is to repeat the following analysis using a neural network.

import numpy as np
import pandas as pd

df1 = pd.read_stata('http://www.stata-press.com/data/r11/auto.dta')

from sklearn import linear_model as lm

X = df1[['weight']].to_numpy()
Y = df1[['mpg']].to_numpy()
lm = lm.LinearRegression()
lm.fit(X, Y)
Y_pred = lm.predict(X)
[lm.intercept_, lm.coef_]

[array([39.44028353]), array([[-0.00600869]])]

The above result (negative slope, of -0.0060) confirms our intuition that fuel efficiency (in terms of miles per gallon, mpg) is lower for heavier vehicles.

Now, let's check out the scatter plot of mpg versus weight, as well as the fitted/predicted regression line.

import matplotlib.pyplot as plt

plt.scatter(X, Y)
plt.xlabel('weight')
plt.ylabel('mpg')
plt.plot(X, Y_pred, color='red')
plt.show()

Your Task

Replicate the above linear regression, $mpg = w_0 + w_1 \cdot weight$, using a simple neural network.

$w_0$ and $w_0$ are generally known as the bias and weight respectively.

Your Exercise:

  1. Repeat with sklearn.MLPRegressor. As before, no hidden layer / activation function. Use "linear / identity" for the activation function if it must be specified. "linear / identity" means that $f(x) = x$.
